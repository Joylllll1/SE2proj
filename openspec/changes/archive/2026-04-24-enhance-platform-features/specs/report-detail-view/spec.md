# Report Detail View

举报详情查看能力，管理员可查看被举报帖子的完整内容。

## ADDED Requirements

### Requirement: 管理员可查看举报帖子详情
管理员在审核举报时 SHALL 能够查看被举报帖子的完整内容。

#### Scenario: 打开帖子详情弹窗
- **WHEN** 管理员点击举报内容的"查看详情"按钮
- **THEN** 系统显示帖子详情弹窗，展示完整帖子信息

#### Scenario: 展示帖子完整信息
- **WHEN** 帖子详情弹窗打开
- **THEN** 弹窗显示帖子标题、内容、标签、情绪、时间、校区等完整信息

### Requirement: 帖子详情弹窗复用现有组件
帖子详情弹窗 SHALL 复用现有的帖子卡片组件和详情页组件样式。

#### Scenario: 样式一致性
- **WHEN** 管理员查看举报帖子详情
- **THEN** 弹窗使用与用户查看帖子详情相同的视觉样式
