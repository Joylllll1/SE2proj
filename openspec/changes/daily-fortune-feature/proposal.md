## Why

每日运势功能为用户增加趣味性和日常互动习惯。通过打卡机制培养用户每日访问平台的习惯，同时提供轻松娱乐的内容体验，增强平台粘性。参考洛谷的日期显示方式，在打卡前保持神秘感，打卡后展示个性化运势内容。

## What Changes

- 新增每日运势打卡功能
  - 打卡前状态：仅显示当前日期（类似洛谷风格）
  - 打卡后状态：显示随机抽取的签文
  - 今日宜/忌：从预设描述库中随机抽取多条建议
- 新增运势历史记录查看
- 新增运势内容库（宜/忌描述、签文）
- UI/UX 设计适配现有的玻璃拟态风格

## Capabilities

### New Capabilities
- `daily-fortune`: 每日运势核心功能，包括打卡、签文抽取、宜忌展示
- `fortune-content`: 运势内容管理，包括签文库、宜忌描述库的存储和随机抽取

### Modified Capabilities
- 无需修改现有 capabilities

## Impact

- **前端**: 新增 `DailyFortune.jsx` 组件，集成到首页或独立路由
- **后端**: 新增运势 API 端点（打卡、获取运势、历史记录）
- **数据存储**: 用户每日打卡记录、运势内容库（初期可前端硬编码，后期迁移至数据库）
- **依赖**: 无新增外部依赖

## 非目标

- 不涉及真实占卜或预测功能，仅为娱乐性质
- 不涉及用户运势数据的社交分享功能（可后续迭代）
- 不涉及付费高级运势内容
