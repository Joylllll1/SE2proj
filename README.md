# NJU树洞

> 面向南京大学师生的半匿名信息交流与情绪表达平台

## 项目简介

NJU树洞致力于为用户提供一个**安全、可控、低压力**的发声空间。用户可以在平台上：

- 发布动态、倾诉情绪
- 分享校园信息
- 寻求帮助
- 通过标签与内容分类实现高效的信息流转
